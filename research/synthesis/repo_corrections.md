# Repository Architecture Update: Strategic Core Corrections

---

## Case 1: IBM Quantum Roadmaps & Fault-Tolerance Announcements (2024–2026)
* **Previous Tag:** Candidate Tech Discontinuity Shock
* **Updated Classification:** **Gray Swan / Anticipated Disruption**
* **Correction Rationale:** While the realization of fault-tolerant quantum computing is a massive paradigm shift, it completely fails the surprise test for industry insiders. IBM's multi-year hardware scaling roadmaps are completely public and systematically updated. Furthermore, national security agencies and global cryptography bodies (like NIST) have spent years actively preparing for this exact vector by mandating and implementing Post-Quantum Cryptography (PQC) standards. This represents a highly telegraphed transition, not a unexpected break.

## Case 2: NVIDIA RTX Spark / Unified Memory Laptops (2026)
* **Previous Tag:** Candidate Tech Discontinuity Black Swan
* **Updated Classification:** **Neither / Endogenous Competitive Response**
* **Correction Rationale:** NVIDIA invading Apple's unified memory moat by shipping high-bandwidth ARM-based SoCs to Windows developer workstations is a major market development, but it does not constitute a Black Swan. It is a text-book endogenous competitive response. When an incumbent (Apple) enjoys highly profitable, uncontested margins among local AI developers due to a specific hardware architecture (M-series unified memory), market equilibrium dictates that rival hardware giants (NVIDIA/Microsoft) will inevitably deploy their R&D to build a matching alternative architecture to reclaim that mindshare.
